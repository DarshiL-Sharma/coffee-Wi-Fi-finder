# ☕ Coffee & Wi-fi Finder – Project Overview

Coffee & Wi-fi Finder is a full-stack web application built using Flask that helps users discover cafes suitable for work. The platform focuses on key factors such as coffee quality, Wi-Fi strength, and power outlet availability, making it easier to find the perfect workspace.

---

## 🎯 Purpose

The goal of this project is to provide a simple and user-friendly way to explore cafes based on essential work-friendly features. It solves a real-world problem for students, freelancers, and remote workers.

---

## ⚙️ Technology Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS (custom styling)
* **Forms:** Flask-WTF
* **Data Handling:** Pandas
* **Database:** CSV file (lightweight storage)
* **Styling:** Custom CSS (Cozy Premium UI theme)
* **Templating:** Jinja2

---

## 🚀 Features

* 📋 View a list of cafes with key details
* ☕ Coffee rating system
* 📶 Wi-Fi strength indicator
* 🔌 Power outlet availability
* 📍 Direct Google Maps location links
* ➕ Add new cafes via form
* 🎨 Clean, modern, and responsive UI
* ✨ Smooth animations and hover effects

---

## 🧠 How It Works

* Cafe data is stored in a CSV file
* Flask reads and displays data dynamically
* Users can submit new cafes via a form
* New entries are appended using Pandas
* Templates render dynamic content using Jinja2
